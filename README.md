# 🍔DAG Task Runner
dag task runner

## Example

1. install dependencies

```
$ npm install
```

2. build

```
$ npm run build
$ npm run build:watch
```

3. run
```
$ node test/apiNode.js
$ node test/webNode.js
```

## support node

- api request
- export file
- script execute
- web access
- delay timer
- element selector
- element events
